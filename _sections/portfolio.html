Photon Mapping (C++)


The following is a ray tracer built from scratch (the only library used was an image reader). The project uses photon mapping to solve the problem of global illumination, sending 'photons' out from light sources and storing them in data structures called KD-trees. The result (as can be seen in the example image below) is that it is able to render colour bleeding and caustics as an addition to standard raytracing.

Brackeys Game Jam (Unity/C#)


A prototype created for the Brackeys game jam within Unity, along with an artist and a sound designer. The premise of the game is that light is a finite resource that is used to be able to see and scare away enemies, and can only be recharged at certain checkpoints.

A list of features added, and my retrospective thoughts on the project can be found on the GitHub readme.

Jurassic World Evolution is a park sim game where you run your own version of Jurassic Park. I worked as a gameplay programmer on the DLC Return To Jurassic Park, implementing features and fixing defects. The title was released on steam to positive reception and can be found here.

Functional C Interpreter and Compiler (C++)


This project is an interpreter and a compiler for a functional C like language that only uses integers and functions as return types (named CMinus).

Lexical scoping is implemented for the language (using nested symbol tables). For syntax examples that show what the program can interpret/compile, see the GitHub page readme.


Unbeatable TicTacToe (C++)


One of my first C++ projects that I created while learning about AI. Uses a MiniMax algorithm with alpha beta pruning for efficiency to create a TicTacToe AI that cannot be beaten.
